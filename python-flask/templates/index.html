WHAT WE DO

The phishing problem is considered a vital issue in industries especially e-banking and e-commerce taking the number of online transactions involving payments. We have identified different features related to legitimate and phishy websites and collected 11056 records from difference sources. When a website is considered SUSPICIOUS that means it can be either phishy or legitimate, meaning the website held some legit and phishy features. Result 1 means legitimate 0 is suspicious -1 is phishing. The main goal of this project is to predicate whether it is suspicious, legitimate or phishy.
